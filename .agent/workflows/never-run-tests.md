# 절대 규칙 (Absolute Rule): 에이전트는 테스트/빌드 명령어를 터미널에서 직접 실행해서는 안 됩니다.

- **[CRITICAL] 터미널 명령어 실행 금지:** 에이전트(AI)는 어떠한 경우에도 `run_command` 기능이나 터미널 도구를 사용하여 로컬 시스템 환경에서 Gradle, Maven, 쉘 스크립트 기반의 `build`, `test`, `compile`을 비롯한 검증 명령어를 실행해서는 안 됩니다.
- **사용자 위임 원칙:** 코드를 수정하거나 검증이 필요한 경우, 에이전트는 사용자에게 실행해야 할 [정확한 명령어]를 마크다운 텍스트 블록으로 안내하고, 사용자가 스스로 프롬프트에 결과를 복사하여 붙여넣고 피드백하도록 기다려야 합니다.
- **자동 완성 금지:** 습관적으로 코드를 검증하려는 백그라운드 프로세싱 시도가 절대 없도록, 해당 행동의 가중치를 0으로 제한합니다.
